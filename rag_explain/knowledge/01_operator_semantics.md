# 算子语义说明（量价因子构建模块）

本知识条目说明本系统因子表达式中各算子的金融含义，用于解读因子的经济逻辑。
来源：项目 operators.py 算子定义 + 量价因子通用语义。

## 时序统计类算子

### Mean(x, N) — N 日滚动均值
对字段 x 取过去 N 个交易日的算术平均。常用于构造均线、平滑噪声。
Mean(close, 20) 即 20 日收盘均价（MA20），代表中期价格中枢；
Mean(volume, N) 代表平均成交量水平，是流动性与活跃度的度量。

### Std(x, N) — N 日滚动标准差
衡量字段 x 在过去 N 日内的波动幅度。Std(close, N) 是价格波动率的代理变量，
是典型的“低波动因子（low volatility）”构建基石。高波动通常对应高风险、
信息不确定性大；波动率因子在 A 股常呈现“低波动溢价”。

### Skew(x, N) / Kurt(x, N) — 偏度 / 峰度
刻画收益分布的非对称性与厚尾程度。正偏度对应右尾（彩票偏好），
学术上“特质偏度”与未来收益负相关；峰度刻画极端事件频率。

### Mad(x, N) — 平均绝对偏离
过去 N 日内 x 偏离其均值的绝对值平均，是对标准差的稳健替代，抗极值。

### Sum(x, N) / Max(x, N) / Min(x, N)
滚动求和 / 最大 / 最小。Max(high,N)、Min(low,N) 常用于构造通道突破、
价格相对位置（如收盘价在 N 日高低点区间的位置 → 反转/动量信号）。

### Quantile(x, N, q) / Med(x, N)
滚动分位数 / 中位数，稳健地刻画 x 的分布位置。

## 时序变化类算子

### Delta(x, N) — N 日变化量
x 当前值减去 N 日前的值，即 x - Ref(x, N)。
Delta(close, N) 是 N 日价格动量/涨跌幅的绝对形式；正值代表上涨。
Delta(volume, N) 代表成交量的放大或萎缩，是量能变化的核心度量。

### Ref(x, N) / Delay — 时间平移
取 N 日前的值。N>0 为历史数据（合法），N<0 会引入未来函数（禁止）。

### Slope(x, N) / Rsquare(x, N) / Resi(x, N) — 滚动时间回归
将 x 对时间 t=1,2,...,N 做回归。Slope 为趋势斜率（动量强度），
Rsquare 为趋势的拟合优度（趋势的“干净”程度），Resi 为残差（短期背离）。
高 Slope + 高 Rsquare = 稳健上升趋势。

### EMA(x, N) / WMA(x, N) — 指数/加权移动平均
对近期数据赋予更高权重的平滑，比简单均值对近期变化更敏感。

## 截面 / 排名类算子

### Rank(x, N) — 滚动时序排名
注意：本系统的 `Rank(x, N)` 是**时序滚动排名**（基于 `df.rolling(N).rank()`），
即对每只股票，在其自身过去 N 日窗口内，对当前值做排名（N=0 时为 expanding 累计排名）。
它衡量的是"当前值在该股票近期历史中的相对高低"，**不是横截面（跨股票）排名**。
排名可消除量纲、抵抗极端值。若需横截面打分，需在外部对截面另行处理。

## 双变量类算子

### Corr(x, y, N) — N 日滚动相关性
x 与 y 在过去 N 日的相关系数。Corr(close, volume, N) 或
Corr(Delta(close,k), Delta(volume,k), N) 是**量价关系**的核心度量：
- 正相关：价涨量增（趋势确认、健康上涨）
- 负相关：价涨量缩 / 价跌量增（量价背离，常预示反转）
量价背离因子（price-volume divergence）是 A 股最有效的因子族之一。

### Cov(x, y, N) — 协方差
未标准化的共同变动度量，与 Corr 类似但保留量纲。

## 算术与变换类算子

### Abs / Sign / Log / Power
绝对值 / 符号 / 对数 / 幂。Log 常用于压缩成交量、成交额的右偏分布；
Abs(Delta(close,1)/close) 是日内涨跌幅绝对值，刻画价格跳动剧烈程度。

### 乘以 -1（取负）
因子表达式末尾 * -1 表示**反向**信号：原本数值大的股票预期收益低。
这通常意味着该因子是“反转型”或“风险规避型”逻辑。
